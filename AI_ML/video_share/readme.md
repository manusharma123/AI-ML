# Screen Sharing ML using FastAPI

## 🚀 Run Project

### 1. Install dependencies
pip install -r requirements.txt

### 2. Start server
uvicorn main:app --reload

### 3. Open in browser
http://localhost:8000/video

## 🧠 ML Integration
Edit `ml_model.py` to plug in your actual model.

## ⚙️ Features
- Real-time screen capture
- FastAPI streaming
- ML-ready pipeline
